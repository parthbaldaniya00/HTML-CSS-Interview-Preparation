# HTML Interview Questions (Top 20)

This file contains the **most commonly asked HTML interview questions**.
These questions test **fundamentals, semantics, accessibility, and real-world understanding**.

---

## 1. What is HTML?
HTML (HyperText Markup Language) is used to structure content on the web by defining elements and their meaning.

---

## 2. What is Semantic HTML?
Semantic HTML uses meaningful tags (like `header`, `nav`, `article`) to describe the purpose of content.

---

## 3. Why is Semantic HTML important?
It improves accessibility, SEO, code readability, and long-term maintainability.

---

## 4. Difference between `div` and `span`
- `div` is a block-level element  
- `span` is an inline element  

---

## 5. Difference between `div`, `section`, and `article`
- `div`: no semantic meaning  
- `section`: groups related content  
- `article`: independent, reusable content  

---

## 6. What is the purpose of `<meta charset="UTF-8">`?
It defines the character encoding so all characters and languages display correctly.

---

## 7. What is the viewport meta tag?
It controls how a webpage scales on mobile devices and enables responsive design.

---

## 8. Difference between `id` and `class`
- `id` is unique  
- `class` is reusable  

---

## 9. Block-level vs Inline elements
- Block elements start on a new line  
- Inline elements stay on the same line  

---

## 10. What are data attributes?
Custom attributes (`data-*`) used to store extra information in HTML elements.

---

## 11. Difference between `<button>` and `<a>`
- `<a>` is for navigation  
- `<button>` is for actions  

---

## 12. What is the `name` attribute in forms?
It is required to send input data to the server.

---

## 13. What is native HTML validation?
Built-in validation using attributes like `required`, `type`, and `pattern`.

---

## 14. Difference between `disabled` and `readonly`
- `disabled` values are not submitted  
- `readonly` values are submitted  

---

## 15. What are meta tags?
They provide metadata such as charset, viewport, and SEO information.

---

## 16. Difference between `async` and `defer`
- `async` loads and executes immediately  
- `defer` executes after HTML parsing  

---

## 17. Can a page have multiple `<header>` or `<footer>` elements?
Yes, they can be used inside sections and articles.

---

## 18. What is the `<main>` tag?
It represents the primary content of the page and must be unique.

---

## 19. What is `<!DOCTYPE html>`?
It tells the browser to use standards mode for rendering the page.

---

## 20. Can HTML work without CSS and JavaScript?
Yes, HTML alone can structure content, create forms, and provide navigation.

---
